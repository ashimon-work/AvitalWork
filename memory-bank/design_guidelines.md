# Design & UX Guidelines

## Golden Rule for All Design Tasks

**ALL** UI/UX design specifications, wireframes, and implementation plans MUST adhere to the principles, color palette, typography, and component styles defined in the official brand and UX strategy document.

This is not optional. It is the single source of truth for ensuring a cohesive, trustworthy, and high-quality user experience across the entire "Global Marketplace App".

*   [`global-marketplace-style-guide.md`](../global-marketplace-style-guide.md)

Before starting any design-related subtask, consult this guide.
